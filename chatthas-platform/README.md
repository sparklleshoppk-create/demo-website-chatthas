# Chattha's Platform — High-Performance Desi Food Ordering

A premium, optimized web application for Chattha's Restaurant, built with Next.js 14, Supabase, and Stripe.

## 🚀 Optimized Architecture

This platform has been audited and optimized for maximum performance:
- **Image Optimization:** Automated compression and lazy loading via `next/image`.
- **Render Optimization:** Context memoization and optimized hook dependency arrays.
- **Animation Performance:** GPU-accelerated Framer Motion transitions with staggered entrance.
- **Dynamic Routing:** App Router with server-side data fetching for minimal JS payload.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Database/Auth:** Supabase
- **Payments:** Stripe & PayFast
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons

## 📦 Getting Started

### 1. Prerequisites
- Node.js 18+ 
- npm / yarn / pnpm

### 2. Installation
```bash
# Install dependencies
npm install
```

### 3. Environment Setup
Copy `.env.example` to `.env.local` and fill in your credentials:
```bash
cp .env.example .env.local
```

### 4. Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 5. Production Build
```bash
npm run build
npm run start
```
